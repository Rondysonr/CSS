# CSS  

---

Para alterar as informações de um paragráfo extruture da seguinte forma o css: 
selecione o seletor ```h1{}``` em seguida o tipo de fonte ```front-family:``` o tamanho: ```front-size:``` e por fim a cor: ```color:```  olhe o exemplo:

---

```
h1{ 
front-family: Arial;
front-size: 20pt;
color: blue;
}
```
